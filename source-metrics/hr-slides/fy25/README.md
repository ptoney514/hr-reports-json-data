# FY25 HR PowerPoint Slides

This directory contains screenshots/exports of PowerPoint slides provided by the HR department that contain data which cannot be calculated from our raw data sources.

## Files to be Added:

1. **faculty-turnover-by-division-source.png**
   - Shows faculty turnover rates by division/college
   - Contains the 13.7% Nursing, 7.5% Pharmacy rates, etc.

2. **voluntary-involuntary-turnover-fy25.png**
   - Shows breakdown of turnover by voluntary/involuntary/retirement
   - Fiscal Year Ending June 30, 2025
   - Separate rates for Faculty, Staff Exempt, and Staff Non-Exempt

3. **faculty-staff-demographic-review-fy25.png**
   - Shows multi-year comparison of turnover rates
   - Includes Higher Education averages vs. Creighton actuals
   - Covers FY23, FY24, and FY25

## Usage Notes:

These images should be saved here when received from the HR department. The data from these slides is manually transcribed into:
- `/src/data/staticData.js` for dashboard display
- `/docs/NON_CALCULABLE_DATA_SOURCES.md` for documentation

## Data Source:
- **Department:** Human Resources
- **Format:** PowerPoint presentation
- **Frequency:** Annual or as updated by HR
- **Last Updated:** January 2025
