## Brand Strategy

### Brand Promise
**"A Jesuit Education at the Intersection of Health and Humanity, Forming Leaders to Build a More Just World."**

### Brand Pillars

1. **The Formative Community** - Diverse paths converging into a transformative journey experienced together
2. **The Body of Scholar-Mentors** - Faculty as attentive teachers and experts dedicated to student success
3. **The Reflective Academic Heart** - Humanist tradition that deepens understanding of ourselves and the world
4. **The Creighton Professional** - Superior professionals who are wholly educated, morally fit, and expertly trained
5. **The Tradition of Impact** - Commitment to service, research, and meaningful contributions

### Brand Personality

- **Steadfastly Driven** - Restless in pursuit of our goals
- **Thoroughly Bright** - Judicious and well-rounded in our intellect
- **Notably Compassionate** - Generous in our care for others
- **Profoundly Curious** - Earnest in our asking of honorable questions

## Visual Identity

### Primary Logo Usage

- **Primary Logo**: Creighton University with dividing line between "Creighton" and "University"
- **Minimum Size**: 1 inch width
- **Clear Space**: 1/3 of logo height on all sides
- **Acceptable Colors**: Creighton Blue (PMS 286), Black, or White only

### Logo Standards

- The single line between "Creighton" and "University" is **required**
- Never stretch, skew, or distort the logo
- Never add visual effects (drop shadows, glows, bevels)
- Never use unapproved fonts to recreate the logo
- Never place on busy backgrounds without sufficient contrast

### Secondary Logos

For schools and colleges:
- Line divides "University" and the school/college name
- Same color restrictions apply
- Cannot be modified or sub-branded with athletic logos

## Typography

### Primary Typeface: Nocturne Serif

**Available Weights:**
- Light, Light Italic
- Regular, Regular Italic  
- Semibold, Semibold Italic
- Bold, Bold Italic

**Usage:** Headlines, display text, formal communications

**Alternatives:**
- Google Font: Gentium Book Basic
- System Font: Georgia

### Secondary Typeface: Proxima Nova

**Available Weights:**
- Light, Light Italic
- Regular, Italic
- Semibold, Semibold Italic
- Bold, Bold Italic
- Extrabold, Extrabold Italic

**Usage:** Body text, captions, user interface elements

**Alternatives:**
- System Font: Arial

### Typography Hierarchy

```css
/* Headlines */
.headline-primary {
  font-family: "Nocturne Serif", "Gentium Book Basic", Georgia, serif;
  font-weight: 600; /* Semibold */
  letter-spacing: -10pt;
}

.headline-secondary {
  font-family: "Nocturne Serif", "Gentium Book Basic", Georgia, serif;
  font-weight: 400; /* Regular */
  font-style: italic;
  letter-spacing: 0;
}

/* Body Text */
.body-text {
  font-family: "Proxima Nova", Arial, sans-serif;
  font-weight: 400; /* Regular */
  letter-spacing: -10pt;
  min-size: 9pt; /* print */ 14pt; /* web */
}

/* Intro Copy */
.intro-copy {
  font-family: "Proxima Nova", Arial, sans-serif;
  font-weight: 700; /* Bold */
  text-transform: uppercase;
  letter-spacing: 50-75pt;
}
```

### Typography Rules

- **No ligatures** in headlines or body copy
- Headlines should NOT always include italic variation - use discretion
- Diamond cross element (◆) can divide copy blocks or emphasize content
- Maintain consistent kerning and spacing

## Photography Guidelines

### Hero Photography Style

**Characteristics:**
- Captured from **low angles** to represent upward trajectory
- **Backlighting** with subjects lit from behind
- **Golden hour** lighting (shortly after sunrise or before sunset)
- **Hopeful, aspirational** mood

**Subject Matter:**
- **Portraiture**: Individual students, faculty, professionals
- **Environmental**: Campus landmarks, cityscapes with people
- **Action**: Students engaged in learning, research, service

### Secondary Photography

**Purpose:** Showcase Creighton's formative community

**Characteristics:**
- **Interpersonal interactions** and collaboration
- **Detail shots** that invite viewers into the scene
- **Standard lighting** and angles
- **Documentary style** capturing authentic moments

### Photo Filters

**Light Leaks Effect:**
- Apply to create "hero style" effect on standard photos
- Use in Photoshop/InDesign with 25-40% opacity
- Set layer to "Overlay" blend mode
- Most effective on images with existing natural light source

## Illustrations & Graphics

### Angled Color Block

- **45° upward angle** from bottom to top
- **Bleed off the page** for forward movement
- **Colors**: Combine PMS 286 and PMS 2767, or use PMS 291/Warm Gray 2C
- Can be used at percentages for text readability

### Diamond and Cross Element ◆

- **Colors**: Any brand color (cross must be white or PMS 2767)
- **Maximum size**: 0.25 inches
- **Usage**: Divide copy blocks or emphasize content
- Apply with intention, not as large design element

### Campus Illustrations

**Style Guidelines:**
- **Line width**: 1-3 pt range consistently
- **Front-facing**, flat depictions (no perspective)
- **Sharp corners** and terminals
- **Symmetrical** architectural renderings when possible
- Mix of organic and geometric shapes

**Available Subjects:**
- St. John's Church
- Creighton Hall details
- The Fountain
- The Flame Sculpture
- Campus archway and bridge
- Billy Bluejay statue

### Stained Glass Graphics

**Purpose:** Capture light, layered nature of the University

**Elements:**
- **Bisecting line** - align shapes at same vertical center
- **Semi-transparent layers** with pathfinder effects
- **Shield and window containers** for detailed icons
- **Duotone effects** with brand colors

### Flame Graphic

**Inspiration:** "The Flame" sculpture by Littleton Alston

**Usage:**
- **45° upward angle**
- **Bleed off bottom** of page
- **Colors**: PMS 2767, PMS 286, PMS 291
- Headlines can overlay if opacity adjusted for readability
- Avoid small text overlays

## Voice & Messaging

### Brand Narrative

*"Nothing worth doing ever goes without effort."*

Creating opportunities for yourself and society, leading people in challenging times, healing the sick and injured in your community—these all require you to give a little extra, to push a little harder, to bring a little fire into the world.

At Creighton University, we supply the oxygen.

### Style Guidelines

1. **Know your audience** - Tailor messaging to specific needs and priorities
2. **Focus on impact** - Communicate results and relevance
3. **Be clear and concise** - Avoid academic jargon and marketing speak
4. **Provide benefits** - Answer "what's in it for the reader?"
5. **Be human and relatable** - Feature student, faculty, and alumni stories
6. **One message well** - Single distinct message per communication
7. **Include call to action** - Clear, motivating next steps
8. **Avoid jargon** - Explain university-specific terms for external audiences

### Voice Characteristics

**Steadfastly Driven:**
- Standard: "In our compassionate community, Creighton students are driven to do more."
- Voiced: "Whatever you do best, do it justice."

**Thoroughly Bright:**
- Standard: "Known for academic excellence, Creighton University is where you'll thrive."
- Voiced: "Nothing worth doing ever goes without effort. So bring your fire into the world. Creighton University will supply the oxygen."

**Notably Compassionate:**
- Standard: "If you want to make a difference in the lives of others, Creighton University is the right place for you."
- Voiced: "For all who believe in creating a better, healthier world—we say: Welcome to Creighton."

**Profoundly Curious:**
- Standard: "Immerse yourself in a campus full of things to do."
- Voiced: "The time you have in college makes a difference. Make it count at Creighton University."

## Digital Implementation

### Web & Digital Colors

Use HEX values for consistency across digital platforms:

```css
:root {
  /* Primary Colors */
  --creighton-blue: #0054A6;
  --creighton-navy: #00245D;
  
  /* Secondary Colors */
  --light-blue: #1F74DB;
  --sky-blue: #95D2F3;
  
  /* Accent Colors */
  --creighton-green: #71CC98;
  --creighton-yellow: #FFC72C;
  
  /* Neutrals */
  --warm-gray-1: #D7D2CB;
  --warm-gray-2: #EBE6E4;
  --light-gray: #F3F3F0;
}
```

### Typography for Web

```css
/* Import fonts */
@import url('https://fonts.googleapis.com/css2?family=Source+Serif+Pro:ital,wght@0,400;0,600;1,400;1,600&display=swap');

/* Fallback for Nocturne Serif */
.font-serif {
  font-family: "Nocturne Serif", "Source Serif Pro", "Gentium Book Basic", Georgia, serif;
}

/* Fallback for Proxima Nova */
.font-sans {
  font-family: "Proxima Nova", "Source Sans Pro", Arial, sans-serif;
}
```

### Accessibility Guidelines

- Maintain 4.5:1 contrast ratio minimum
- Include focus indicators on interactive elements
- Use semantic HTML structure
- Provide alt text for all decorative and informational images
- Ensure keyboard navigation functionality
